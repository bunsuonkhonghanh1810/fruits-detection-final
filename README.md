## 🚀 Cách chạy trên máy cục bộ

Chỉ cần **copy & paste toàn bộ đoạn dưới đây** vào terminal:

```bash

# 2️⃣ Cài đặt thư viện cần thiết
pip install --upgrade pip && \
pip install -r requirements.txt

# 3️⃣ Chạy ứng dụng
streamlit run app.py
